烴類

烷 · 烯 · 炔 · 苯 ─ 命名、反應、製備、鑑別

IUPAC 命名完整規則
1

找最長碳鏈 → 決定母名

1碳=甲(meth) 2碳=乙(eth) 3碳=丙(prop) 4碳=丁(but) 5碳=戊(pent)
6碳=己(hex) 7碳=庚(hept) 8碳=辛(oct) 9碳=壬(non) 10碳=癸(dec)

2

從靠近取代基/官能基的一端開始編號

烯炔：官能基位置優先（不一定從取代基端）
兩端距離相同 → 選取代基數目最多的方向 → 再選位置最小

3

列出取代基：按字母順序，相同加 di-/tri-/tetra-

常見取代基：甲基(methyl)、乙基(ethyl)、丙基(propyl)、苯基(phenyl)
注意：di-/tri- 不計入字母順序，如 ethyl 在 dimethyl 前面

4

組合命名

格式：取代基位置-取代基名稱 + 主鏈名稱 + 官能基字尾（位置）
例：2-methylbutane、3-ethyl-2-methylpentane、but-2-ene

⚠ 順反異構物命名：
傳統：cis-（相同基同側）/ trans-（相同基異側）
IUPAC：Z-（優先基同側）/ E-（優先基異側）— 按 CIP 優先順序判斷
物理性質
碳數常溫狀態用途
C1–C4氣體天然氣（CH₄）、液化石油氣
C5–C17液體汽油、煤油、柴油
C18+固體蠟燭（石蠟）、潤滑油
化學反應
① 燃燒反應（完全燃燒）
CₙH₂ₙ₊₂ + (3n+1)/2 O₂ → nCO₂ + (n+1)H₂O

例：CH₄ + 2O₂ → CO₂ + 2H₂O
例：C₂H₆ + 7/2 O₂ → 2CO₂ + 3H₂O
⚠ 不完全燃燒 → 生成CO（有毒）或碳粒（黑煙）
碳氫比越高，燃燒越容易不完全（苯的黑煙最明顯）
② 鹵化取代反應（需 hν 光照或高溫，自由基機制）
CH₄ + Cl₂ →(hν)→ CH₃Cl + HCl　（氯甲烷）
CH₃Cl + Cl₂ →(hν)→ CH₂Cl₂ + HCl　（二氯甲烷）
CH₂Cl₂ + Cl₂ →(hν)→ CHCl₃ + HCl　（三氯甲烷/氯仿）
CHCl₃ + Cl₂ →(hν)→ CCl₄ + HCl　（四氯化碳）
📌 自由基鏈鎖機制三步驟：
引發（Initiation）：Cl₂ →(hν)→ 2Cl•
增長（Propagation）：Cl• + CH₄ → CH₃• + HCl；CH₃• + Cl₂ → CH₃Cl + Cl•
終止（Termination）：兩個自由基結合
❌ 烷類不與 KMnO₄、溴水反應（不褪色）→ 用來排除烯炔的關鍵依據！
結構特徵
・C=C 雙鍵：1條σ鍵 + 1條π鍵
・雙鍵碳：sp²混成，鍵角120°
・雙鍵4個原子共平面
・π鍵不能旋轉 → 有順反異構物
順反異構物條件：
雙鍵兩端的碳，各自連接的兩個取代基不同
若某端接兩個相同基團 → 無順反異構物
加成反應（最主要）
反應名稱試劑/條件反應式備注
催化氫化H₂ / Ni, ΔCH₂=CH₂ + H₂ → CH₃-CH₃工業氫化植物油
加溴（鑑定）Br₂/CCl₄ 或 Br₂/H₂OCH₂=CH₂ + Br₂ → CH₂Br-CH₂Br橙紅→無色（褪色）
加HX（馬可尼可夫）HBr、HCl、HICH₃CH=CH₂ + HBr → CH₃CHBr-CH₃（主）H加含H多的碳
水合（製醇）H₂O / H⁺, ΔCH₂=CH₂ + H₂O → CH₃CH₂OH工業製乙醇
加次氯酸HOCl / H₂OCH₂=CH₂ + HOCl → HOCH₂CH₂Cl製環氧乙烷中間步
氧化反應
KMnO₄（冷稀，中性/鹼性）：
3CH₂=CH₂ + 2KMnO₄ + 4H₂O → 3HOCH₂CH₂OH + 2MnO₂↓ + 2KOH
現象：紫色褪去，生棕色MnO₂沉澱
KMnO₄（熱濃酸性）— 雙鍵斷裂：
雙鍵碳有2個H → 生成 CO₂ + H₂O
雙鍵碳有1個H → 生成 RCOOH（羧酸）
雙鍵碳有0個H → 生成 RCOR'（酮）
🔑 熱濃KMnO₄可用來推斷烯類的結構（從氧化產物逆推）
製備烯類的方法
① 醇脫水（消去反應）：CH₃CH₂OH →(濃H₂SO₄, 170°C)→ CH₂=CH₂ + H₂O
② 鹵烷消去（鹼）：CH₃CH₂Br + KOH →(醇，Δ)→ CH₂=CH₂ + KBr + H₂O
③ 裂解（工業）：石油裂解 → 乙烯、丙烯等小分子烯類
聚合反應
加聚：nCH₂=CH₂ → (-CH₂-CH₂-)ₙ　（聚乙烯 PE）
nCH₂=CHCl → (-CH₂-CHCl-)ₙ　（聚氯乙烯 PVC）
結構特徵
・C≡C 三鍵：1條σ鍵 + 2條π鍵
・炔碳：sp混成，鍵角180°
・直線形
・末端炔的 C-H 有微弱酸性
酸性比較：
末端炔的 ≡C-H 酸性 > H₂O
但弱於醇（醇弱於水）
→ 末端炔可被強鹼去質子
反應試劑/條件反應式備注
加氫（第1次）H₂ / Pd（控制）HC≡CH + H₂ → H₂C=CH₂Lindlar催化劑可停在烯
加氫（第2次）H₂ / NiH₂C=CH₂ + H₂ → H₃C-CH₃完全氫化
加溴（第1次）Br₂/CCl₄HC≡CH + Br₂ → CHBr=CHBr褪色
加溴（第2次）Br₂/CCl₄（過量）CHBr=CHBr + Br₂ → CHBr₂-CHBr₂可加成兩次
加HCl（製PVC單體）HCl / HgCl₂催化HC≡CH + HCl → CH₂=CHCl（氯乙烯）工業重要反應
加水（製乙醛）H₂O / HgSO₄, H₂SO₄HC≡CH + H₂O → CH₃CHO（乙醛）先生成烯醇，再互變
末端炔 + 銀氨液[Ag(NH₃)₂]⁺RC≡CH + [Ag(NH₃)₂]⁺ → RC≡CAg↓ + NH₄⁺白色沉澱（鑑定）
末端炔 + 亞銅氨液[Cu(NH₃)₂]⁺RC≡CH + [Cu(NH₃)₂]⁺ → RC≡CCu↓ + NH₄⁺磚紅色沉澱（鑑定）
製備炔類的方法
① 工業製乙炔：CaC₂ + 2H₂O → HC≡CH + Ca(OH)₂
（電石與水反應，電石法）
② 鹵烷雙消去：CH₃CHBr₂ + 2KOH →(醇，Δ)→ HC≡CH + 2KBr + 2H₂O
❌ 非末端炔（如 CH₃-C≡C-CH₃）不與銀氨、亞銅氨反應！
→ 這是鑑別末端炔與非末端炔的方法
苯的結構特徵
・正六邊形，所有原子共平面
・C-C鍵長 139pm（單154，雙134之間）
・鍵角 120°，sp²混成
・6個π電子離域共振 → 高度穩定
・主要反應：取代（不易加成）
定位效應
鄰對位導向（活化）：-OH, -NH₂, -R
鄰對位導向（鈍化）：-X（鹵素）
間位導向（鈍化）：-NO₂, -COOH, -CHO, -SO₃H
燃燒時有明顯黑煙（碳氫比高）
反應條件方程式產物特性
鹵化FeCl₃ 或 FeBr₃C₆H₆ + Cl₂ → C₆H₅Cl + HCl氯苯（無色液體）
硝化濃HNO₃ + 濃H₂SO₄，50-60°CC₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O硝基苯（淡黃色油狀，有毒）
磺化（可逆）發煙H₂SO₄，ΔC₆H₆ + H₂SO₄ ⇌ C₆H₅SO₃H + H₂O苯磺酸（加水可逆回苯）
Friedel-Crafts烷基化RCl / AlCl₃C₆H₆ + CH₃Cl → C₆H₅CH₃ + HCl甲苯
Friedel-Crafts醯基化RCOCl / AlCl₃C₆H₆ + CH₃COCl → C₆H₅COCH₃ + HCl苯乙酮
催化加氫（強烈）H₂ / Ni，高溫高壓C₆H₆ + 3H₂ → C₆H₁₂環己烷
燃燒點燃2C₆H₆ + 15O₂ → 12CO₂ + 6H₂O明顯黑煙
📌 甲苯的特殊反應：
甲苯 + KMnO₄（酸性）→ 苯甲酸（苯環側鏈被氧化）
→ 苯本身不與KMnO₄反應，但甲苯可以！

醇 · 酚 · 醚

命名、製備、反應、鑑別——三者連貫學習

三者核心比較表 必考
性質醇 ROH酚 ArOH醚 ROR'
-OH 位置脂肪碳苯環碳無 -OH
酸性極弱（弱於水）弱酸（酸性 > H₂CO₃）中性
與 Na 金屬✅ 產生 H₂✅ 產生 H₂❌
與 NaOH❌ 不反應✅ 溶解（中和）❌
與 Na₂CO₃❌✅ 溶解 + CO₂↑❌
FeCl₃ 溶液無顯著變色✅ 紫色—
KMnO₄1°/2°褪色；3°不褪褪色不褪色
溴水一般不反應✅ 白色沉澱（三溴苯酚）不反應
K₂Cr₂O₇/H⁺1°/2°橙→綠；3°不變—不反應
與 HXROH + HX → RX + H₂O不易不易
🔑 快速區分三者：
醇 vs 酚 → 加 NaOH（酚溶解，醇不反應）或加 FeCl₃（酚紫色）
醇 vs 醚 → 加 Na 金屬（醇放H₂，醚不反應）
酚 vs 羧酸 → 加 Na₂CO₃（羧酸放CO₂，酚溶解但無CO₂）
1

烯類水合（加成反應）

CH₂=CH₂ + H₂O →(H⁺, Δ)→ CH₃CH₂OH
工業上大量製造乙醇的方法，需磷酸催化，300°C，70atm

2

鹵烷水解

RX + NaOH →(H₂O, Δ)→ ROH + NaX
SN2機制（一級鹵烷）或 SN1機制（三級鹵烷）

3

醛/酮還原

RCHO + H₂ →(Ni)→ RCH₂OH（1°醇）
RCOR' + H₂ →(Ni)→ RCHOHR'（2°醇）

4

澱粉/糖發酵（生物法製乙醇）

C₆H₁₂O₆ →(酵母菌)→ 2C₂H₅OH + 2CO₂
葡萄糖在無氧條件下發酵，製造酒精

📌 醇的級別決定氧化產物（這是考試最常考的點）
反應試劑/條件反應式備注
與Na（活潑金屬）Na2ROH + 2Na → 2RONa + H₂↑速率：醇 < 水 < 酸
氧化（1°→醛）KMnO₄（控制量）RCH₂OH → RCHO需控制，否則繼續氧化
氧化（1°→酸）KMnO₄（過量）RCH₂OH → RCOOH直接到羧酸
氧化（2°→酮）KMnO₄ 或 K₂Cr₂O₇RCHOHR' → RCOR'K₂Cr₂O₇：橙→綠（Cr⁶⁺→Cr³⁺）
氧化（3°）KMnO₄ 或 K₂Cr₂O₇不反應K₂Cr₂O₇ 顏色不變（酒測原理）
消去（高溫成烯）濃H₂SO₄，170°CCH₃CH₂OH → CH₂=CH₂ + H₂O分子內脫水
取代（低溫成醚）濃H₂SO₄，140°C2CH₃CH₂OH → CH₃CH₂OCH₂CH₃ + H₂O分子間脫水
酯化RCOOH + 濃H₂SO₄, ΔRCOOH + R'OH ⇌ RCOOR' + H₂O可逆；酸脫OH，醇脫H
與HXHI > HBr > HClROH + HX → RX + H₂O3° > 2° > 1°醇活性
⭐ 記憶口訣：
高溫(170°C)→消去→烯；低溫(140°C)→縮合→醚
⭐ 酒精測試原理：
C₂H₅OH + K₂Cr₂O₇ + H₂SO₄ → CH₃COOH + Cr₂(SO₄)₃
橙色 K₂Cr₂O₇ → 綠色 Cr₂(SO₄)₃
製備苯酚
工業法（異丙苯法）：苯 + 丙烯 →(AlCl₃)→ 異丙苯 →(O₂,H⁺)→ 苯酚 + 丙酮
實驗室：苯磺酸鈉 + NaOH →(熔融)→ 苯酚鈉 →(H⁺)→ 苯酚
酚的反應
反應試劑方程式現象
酸鹼（弱酸）NaOHC₆H₅OH + NaOH → C₆H₅ONa + H₂O苯酚溶解（苯酚微溶→苯酚鈉可溶）
與Na₂CO₃Na₂CO₃2C₆H₅OH + Na₂CO₃ → 2C₆H₅ONa + H₂O + CO₂↑放出CO₂（酚酸性 > H₂CO₃）
與NaNa2C₆H₅OH + 2Na → 2C₆H₅ONa + H₂↑放H₂
溴化（不需催化劑）Br₂/H₂O（溴水）C₆H₅OH + 3Br₂ → C₆H₂Br₃OH↓ + 3HBr白色沉澱（三溴苯酚）
FeCl₃ 顯色FeCl₃(aq)形成紫色錯合物紫色（靈敏鑑定）
⚠ 酚與溴水的反應不需要FeBr₃催化（不同於苯）
因為-OH是強力鄰對位活化基，使苯環極為活化
⚠ 苯酚鈉（C₆H₅ONa）加酸（如H₂CO₃）→ 苯酚重新沉澱
因為苯酚酸性 < H₂CO₃ 的反向說明：弱酸可從弱酸鹽中被強酸置換出

醛 · 酮

製備、氧化、還原、鑑別——含銀鏡反應完整機制

醛 vs 酮 核心差異 必考
性質醛 RCHO酮 RCOR'
羰基位置鏈端（接至少1個H）鏈中（兩側都接碳）
多倫試劑（銀鏡）✅ 有銀鏡❌ 無
斐林試劑（磚紅）✅ 磚紅沉澱（脂肪醛）❌ 無
KMnO₄褪色不易褪色
K₂Cr₂O₇/H⁺橙→綠不易反應

⚠ 特殊情況：
苯甲醛（C₆H₅CHO）：可與多倫反應 ✅，但不與斐林試劑反應 ❌
甲醛（HCHO）：與多倫、斐林都反應 ✅（甲醛銀鏡反應特別靈敏）
甲酸（HCOOH）：含醛基，也與兩者都反應 ✅
1

一級醇氧化（最常考）

RCH₂OH →(KMnO₄，控制量)→ RCHO
注意：KMnO₄過量會繼續氧化成羧酸

③ 加入乾燥劑（如無水CaCl₂）吸水

與HClRNH₂ + HCl → RNH₃⁺Cl⁻（銨鹽）鹼性，易中和

1

羧酸 + 胺（縮合，需高溫）

RCOOH + R'NH₂ → [RCOONH₃R'] →(Δ, 150-200°C)→ RCONHR' + H₂O
先形成銨鹽，再加熱脫水成醯胺
條件較嚴苛，但原料便宜易取得

2

酯 + 胺（胺解，溫和）解答的方法

還原性判斷核心原則 必考
📌 關鍵：有無游離的醛基（-CHO）或酮基（-CO-）
若羰基被封住（與另一個糖單元相連）→ 無還原性
糖組成連接方式游離醛/酮基還原性斐林/多倫
葡萄糖單醣—游離醛基✅ 有✅
果糖單醣—游離酮基✅ 有✅
半乳糖單醣—游離醛基✅ 有✅
麥芽糖葡萄糖+葡萄糖1→4連接，一端有游離醛基有游離醛基✅ 有✅
乳糖半乳糖+葡萄糖1→4連接，葡萄糖端有游離醛基有游離醛基✅ 有✅
蔗糖葡萄糖+果糖1→2連接：醛基＋酮基互相連接無游離羰基❌ 無❌
澱粉多個葡萄糖1→4（直鏈）/ 1→6（分支）末端才有，極少❌ 幾乎無❌
纖維素多個葡萄糖β-1→4連接極少❌❌
🔑 記憶訣：蔗糖是唯一的雙醣非還原糖！
因為葡萄糖的C1（醛基端）和果糖的C2（酮基端）互相連接，兩端都被封住了
麥芽糖和乳糖都是1→4連接，1端被封但4端（葡萄糖）仍有游離醛基
📌 蔗糖水解驗證：
蔗糖水溶液 + 稀H₂SO₄, Δ → 葡萄糖 + 果糖（兩者都有還原性）
→ 水解後加斐林試劑 → 磚紅色沉澱出現
→ 確認蔗糖已水解
性質澱粉纖維素
單體α-葡萄糖β-葡萄糖
連接方式α-1,4（直鏈）＋α-1,6（支鏈）β-1,4（直鏈）
結構螺旋狀，可溶於熱水直鏈，緊密排列，不溶於水
與碘液✅ 藍紫色❌ 不反應
水解產物麥芽糖→葡萄糖纖維二糖→葡萄糖
人體消化✅ 可消化（澱粉酶）❌ 不可消化（無β-1,4糖苷酶）
用途食物、食用澱粉紙張、棉花、木材
📌 碘液遇澱粉變藍紫色的原因：
I₂分子進入澱粉的螺旋結構內部，形成澱粉-碘錯合物，呈藍紫色
加熱後螺旋打開 → 藍色消失（冷卻後復原）

Q1. 下列有機物中，哪一種同時可以與多倫試劑和斐林試劑反應？
乙醛是脂肪醛（非苯甲醛），可與多倫試劑（產生銀鏡）和斐林試劑（磚紅色Cu₂O沉澱）都反應。
❌ 丙酮是酮，兩者都不反應。
❌ 苯甲醛可與多倫反應，但不與斐林反應（芳香醛特例）。
❌ 蔗糖是非還原糖，兩者都不反應（水解後才有還原性）。
Q2. 乙醇（CH₃CH₂OH）在濃H₂SO₄催化下加熱，140°C 與 170°C 的主要產物分別是？
140°C：分子間脫水（取代反應）→ 乙醚（CH₃CH₂OCH₂CH₃）
170°C：分子內脫水（消去反應）→ 乙烯（CH₂=CH₂）
口訣：低溫成醚（兩分子相遇），高溫成烯（一分子自己脫水）
Q3. 聚合物 (-OOC-C₆H₄-COO-CH₂CH₂-)ₙ 的單體是？
找酯鍵 -COO-，切斷：
-OOC-C₆H₄-COO- 這段 → 切斷後補-OH → HOOC-C₆H₄-COOH（對苯二甲酸）
-CH₂CH₂- 這段 → 切斷後補-OH → HO-CH₂CH₂-OH（乙二醇）
→ 這是 PET！（寶特瓶材料）
Q4. 下列敘述關於蔗糖與麥芽糖的比較，哪個正確？
蔗糖：葡萄糖C1（醛基端）與果糖C2（酮基端）互相連接 → 兩端都封住 → 非還原糖
麥芽糖：兩個葡萄糖以1→4連接，C1被封但另一端葡萄糖的C1仍有游離醛基 → 還原糖
蔗糖是所有常見雙醣中唯一的非還原糖！
Q5. 如何用一種試劑，一次區分苯酚、乙醛、蔗糖三種物質？
斐林試劑：
乙醛（脂肪醛）→ 磚紅色沉澱
苯酚 → 無反應（酚不與斐林反應）
蔗糖 → 無反應（非還原糖）
→ 三者現象完全不同，可一次區分！

❌ FeCl₃：苯酚紫色，但乙醛和蔗糖都無明顯差異
❌ KMnO₄：苯酚和乙醛都褪色，無法區分這兩個
Q6. 耐綸-66（Nylon-66）的兩種單體是？為什麼叫「66」？
耐綸-66：H₂N-(CH₂)₆-NH₂（己二胺，6個碳）+ HOOC-(CH₂)₄-COOH（己二酸，6個碳）
「66」= 第一個6代表己二胺的碳數，第二個6代表己二酸的碳數

耐綸-6（不同！）：己內醯胺（一種單體，開環聚合）

鍵型：醯胺鍵 -CO-NH-（肽鍵），縮聚反應，脫出H₂O
Q7. 苯酚（C₆H₅OH）與乙醇（CH₃CH₂OH）可用哪種試劑最有效地鑑別？
FeCl₃：苯酚 → 紫色；乙醇 → 無顏色變化。最快速靈敏。
❌ Na金屬：兩者都放H₂，無法區分。
❌ K₂Cr₂O₇：乙醇（1°醇）會使橙色變綠色；苯酚也會被氧化使KMnO₄褪色，但K₂Cr₂O₇與苯酚反應不明顯，較難判斷。
❌ 多倫試劑：兩者都不含醛基，不反應。
Q8. 下列關於酯化反應的敘述，何者正確？（選出所有正確的）
✅ B正確：機制為酸脫-OH，醇脫-H，水來自羧酸的羥基
❌ A錯：酯化反應是可逆的（⇌），加水可水解回去
❌ C錯：濃H₂SO₄只是催化劑，只加快速率，不改變平衡位置和K值
❌ D錯：催化劑不影響平衡，不能提高產率

① 增加酸或醇的量 ② 移除水（乾燥劑） ③ 蒸餾移除酯
