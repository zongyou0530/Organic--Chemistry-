醛 vs 酮　核心差異
性質醛 RCHO酮 RCOR'
羰基位置鏈端（至少接1個H）鏈中（兩端都接碳）
多倫試劑（銀鏡）✅ 有銀鏡❌ 無
斐林試劑（磚紅）✅ 磚紅沉澱（脂肪醛）❌ 無

⚠ 苯甲醛特殊：可與多倫反應，但不與斐林反應！
⚠ 甲酸（HCOOH）含醛基，兩種試劑都能反應！
兩種試劑的反應方程式 ⭐⭐⭐
多倫試劑（銀鏡反應）：
RCHO + 2[Ag(NH₃)₂]⁺ + 2OH⁻ → RCOO⁻ + 2Ag↓ + 4NH₃ + H₂O
現象：試管壁出現光亮銀鏡
斐林試劑：
RCHO + 2Cu²⁺ + 4OH⁻ →(加熱)→ RCOOH + Cu₂O↓ + 2H₂O
現象：深藍色 → 磚紅色沉澱（Cu₂O）
1

一級醇氧化（最常考）

RCH₂OH + KMnO₄（控制量）→ RCHO → 若繼續氧化 → RCOOH

2

炔類水合（工業）

HC≡CH + H₂O →(HgSO₄, H₂SO₄)→ CH₃CHO（乙醛）

二級醇氧化：(CH₃)₂CHOH + K₂Cr₂O₇ → (CH₃)₂C=O（丙酮）
Friedel-Crafts醯基化：C₆H₆ + CH₃COCl →(AlCl₃)→ C₆H₅COCH₃（苯乙酮）

羧酸・酯

酸性比較・酯化・水解・皂化

羧酸的反應
反應方程式備注
與NaOHRCOOH + NaOH → RCOONa + H₂O酸鹼中和
與Na₂CO₃2RCOOH + Na₂CO₃ → 2RCOONa + H₂O + CO₂↑產生CO₂氣泡
與Na2RCOOH + 2Na → 2RCOONa + H₂↑產生H₂
酯化（⭐）RCOOH + R'OH ⇌ RCOOR' + H₂O可逆，濃H₂SO₄催化
甲酸特殊HCOOH →(濃H₂SO₄)→ CO + H₂O甲酸含醛基，可銀鏡/斐林
🔑 酯化機制：酸脫-OH，醇脫-H（¹⁸O同位素追蹤實驗驗證）
取代基影響酸性：吸電子基（-Cl、-NO₂）→ 酸性增強；供電子基（-CH₃）→ 酸性減弱
例：CHCl₂COOH > ClCH₂COOH > CH₃COOH
酯的水解 — 酸性 vs 鹼性
酸性水解鹼性水解（皂化）
條件H⁺，加熱NaOH，加熱
產物羧酸 + 醇羧酸鹽 + 醇
可逆性✅ 可逆（⇌）❌ 不可逆（→）
原因—羧酸鹽不會再與醇反應
油脂皂化：油脂（三酸甘油酯）+ 3NaOH → 甘油 + 3脂肪酸鈉（肥皂）
乙酸乙酯：K ≈ 4（CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O）
提高酯化產率的方法：
① 增加酸或醇的量（移動平衡）
② 移除產物（蒸出酯或水，乾燥劑吸水）
③ 加催化劑（濃H₂SO₄，只加快速率，不改變平衡）

胺・醯胺

鹼性・製備・縮合

胺類
鹼性強弱：NaOH > 三級胺 > 二級胺 > 一級胺 > NH₃ > 苯胺
反應方程式
與HClRNH₂ + HCl → RNH₃⁺Cl⁻
苯胺 + 溴水C₆H₅NH₂ + 3Br₂ → C₆H₂Br₃NH₂↓（白色）+ 3HBr
醯胺的製備（三種方式）⭐
1

羧酸 + 胺（縮合）

RCOOH + R'NH₂ →(Δ)→ RCONHR' + H₂O
需高溫（150-200°C）

2

酯 + 胺（胺解）⭐ 解答的方式

RCOOR'' + R'NH₂ → RCONHR' + R''OH
溫和條件即可，產率較高，-OR''為離去基

3

醯氯 + 胺（最快，大學常用）

RCOCl + R'NH₂ → RCONHR' + HCl
低溫即可，反應劇烈，副產物HCl

🔑 三種方法活性比較：醯氯 > 酯 > 羧酸
分科測驗只需掌握方法1和2，醯氯法偏大學範圍

聚合物

加聚・縮聚・反推單體・計算

加聚 vs 縮聚　核心比較
加成聚合（加聚）縮合聚合（縮聚）
單體條件含不飽和鍵（C=C, C≡C）含兩種官能基（如-OH和-COOH）
脫出小分子❌ 無（原子全保留）✅ 有（H₂O, HCl…）
分子式關係聚合物 = n × 單體聚合物 ≠ n × 單體
反推單體方法每兩C切一刀，加回雙鍵找酯鍵/醯胺鍵，切斷後各補官能基
加聚：反推單體（常考！）
📌 方法：把主鏈每兩個碳切一刀，加回雙鍵
聚合物單體縮寫用途
(-CH₂-CH₂-)ₙCH₂=CH₂PE塑膠袋
(-CH₂-CHCl-)ₙCH₂=CHClPVC水管、電線皮
(-CH₂-CHCH₃-)ₙCH₂=CHCH₃PP容器、繩索
(-CF₂-CF₂-)ₙCF₂=CF₂PTFE 鐵氟龍不沾鍋
(-CH₂-CH(CN)-)ₙCH₂=CHCNPAN合成纖維
(-CH₂-C(CH₃)(COOCH₃)-)ₙCH₂=C(CH₃)COOCH₃PMMA 壓克力玻璃替代
縮聚：反推單體方法 ⭐
🔑 切斷規則：
酯鍵 -CO-O-　→　左邊補 -COOH，右邊補 HO-
醯胺鍵 -CO-NH-　→　左邊補 -COOH，右邊補 H₂N-
聚合物單體1單體2脫去用途
聚酯 PETHOCH₂CH₂OH（乙二醇）HOOC-C₆H₄-COOH（對苯二甲酸）H₂O寶特瓶
耐綸-66H₂N-(CH₂)₆-NH₂（己二胺）HOOC-(CH₂)₄-COOH（己二酸）H₂O尼龍
耐綸-6己內醯胺（開環）——尼龍
蛋白質胺基酸（含-NH₂和-COOH）H₂O生物體
⚠ 耐綸-66的「66」= 兩種單體都是6個碳，不要搞混
加聚：M(聚合物) = n × M(單體)
縮聚（n很大時近似）：M(聚合物) ≈ n × [M(單體) − M(脫出小分子)]
例：PET，乙二醇(62) + 對苯二甲酸(166) - H₂O(18) = 192 per repeat unit
若M(PET) = 19200，則 n = 19200/192 = 100

Q1. 下列哪種試劑可以區分 乙醛（CH₃CHO）和 丙酮（CH₃COCH₃）？
乙醛含醛基，可與斐林試劑反應產生磚紅色沉澱（Cu₂O）。丙酮是酮，不與斐林試劑反應。
❌ 溴水和KMnO₄兩者都能被醛和大多數酮還原，無法區分。
❌ Na金屬：兩者都不含活潑氫，不反應。
Q2. 乙醇在濃H₂SO₄、170°C 的條件下，主要產物是？
關鍵口訣：高溫(170°C)消去成烯，低溫(140°C)取代成醚。
170°C → 分子內脫水 → 乙烯（消去反應）
140°C → 分子間脫水 → 乙醚（取代反應）
Q3. 蔗糖與麥芽糖的主要差異是？
蔗糖：葡萄糖的醛基＋果糖的酮基互相連接 → 兩端都封住 → 非還原糖，不與斐林/多倫反應。
麥芽糖：兩個葡萄糖，其中一端仍有游離醛基 → 還原糖，可與斐林/多倫反應。
記憶：蔗糖是唯一的雙醣非還原糖！
Q4. 下列關於酯化反應的敘述，哪個正確？
酯化機制：酸脫-OH，醇脫-H → 脫水縮合。
用¹⁸O同位素追蹤實驗：若在醇的-OH上標記¹⁸O，水分子中會含¹⁸O，酯中不含 → 確認水來自酸的-OH + 醇的-H。
反應是可逆的（⇌），濃H₂SO₄只是催化劑。
Q5. 苯酚（C₆H₅OH）與乙醇（C₂H₅OH）最有效的鑑別方法是？
FeCl₃溶液：苯酚 → 紫色；乙醇 → 無明顯顏色變化。最靈敏也最直接。
❌ Na金屬：兩者都會產生H₂，無法區分。
❌ KMnO₄：乙醇（1°醇）會褪色，苯酚也會，難以區分。
⚠ 溴水也可以（苯酚→白色沉澱，乙醇不反應），但FeCl₃更快速靈敏。
Q6. 聚合物 (-CO-(CH₂)₄-CO-NH-(CH₂)₆-NH-)ₙ 的單體是？
找醯胺鍵 -CO-NH- → 切斷 → 左邊補-COOH，右邊補H₂N-
左段：HOOC-(CH₂)₄-COOH = 己二酸（6個碳的二元酸）
右段：H₂N-(CH₂)₆-NH₂ = 己二胺（6個碳的二元胺）
→ 這是 耐綸-66（Nylon-66）！
